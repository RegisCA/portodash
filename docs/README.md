# Documentation Organization

This folder contains supplementary documentation organized by category.

## Structure

### archive/
Historical implementation notes and outdated guides preserved for reference.

### proposals/
Proposal documents for potential future enhancements and optimizations.

## Main Documentation

Primary documentation is located in the repository root:

- **README.md** - User guide and getting started
- **CONTRIBUTING.md** - Contributing guidelines
- **CHANGELOG.md** - Version history
- **DEVELOPMENT.md** - Development history and architecture
- **ACCESSIBILITY.md** - Accessibility features and guidelines
- **UX_DESIGN.md** - Design system documentation
- **UX_SUMMARY.md** - UX enhancement release notes
- **UX_CHANGELOG.md** - Detailed UX change log
- **UX_QUICKREF.md** - Quick reference for UX improvements
- **YFINANCE_OPTIMIZATION.md** - Technical guide for data fetching

## Design Specs

Design specifications, screenshots, and references are in:
- **specs/** - Design specs, screenshots, mockups, and UX references
